# Fieldcraft V1 Example Scenarios

These are small reference scenarios for the v1 authoring shape. They are not tutorials, campaigns, or rule-bearing sample games.

- `bridgehead-square.fieldcraft.json` demonstrates a square-grid tabletop objective layout with sides, marker art, facing, styling, properties, stacked markers, a board background image, and an audio asset kept in the package.
- `ridgeline-hex.fieldcraft.json` demonstrates a pointy-top hex board with opposing sides, unassigned objectives, marker art, facing, styling, and descriptive properties.
- `convoy-free-coordinate.fieldcraft.json` demonstrates a free-coordinate board with non-zero bounds, continuous positions, sides, marker art, facing, styling, and descriptive properties.

The `assets/` directory is package-local media referenced by these files. The image and audio assets are deterministic fixtures so the examples stay small and easy to keep under version control.
