Trusted local fixture media for package-asset testing.

- `assets/checkerboard-32.png` is a generated 32x32 checkerboard PNG.
- `assets/marker-alpha.png`, `assets/marker-bravo.png`, `assets/marker-command.png`, and `assets/marker-objective.png` are generated 96x96 PNG marker-art fixtures with text labels for manual import and assignment checks.
- `assets/test-tone-440hz.wav` is a generated short mono WAV tone.

These files are intentionally small and deterministic so desktop asset import can be tested without pulling external media into the repo.
